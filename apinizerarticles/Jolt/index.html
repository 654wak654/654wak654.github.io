Jolt ile JSON Veri Dönüşümü (Transformation)

Nedir Jolt

Jolt, Java bazlı bir "JSON'dan JSON'a dönüşüm" kütüphanesidir. Ancak Java bazlı olması bizim için önemli değil çünkü dönüşümler çoğu zaman Java kullanmaya gerek kalmadan Jolt'un kendi dönüşüm tanımlama formatı ile yapılabiliyor. Jolt'un asıl amacıda zaten bu. Yine bir JSON dosyası ile tanımlanan bu dönüşüm "dilini" kullanarak verilen JSON verisini belittiğimiz gibi biçimlendirerek istediğimiz yapıya dönüştürebiliyoruz. Jolt'un kendi içinde hali hazırda bulunmayan operasyonlar içinde Jolt kütüphanesini genişletmek çok kolay.

Neden Jolt

Jolt olmadan dönüşüm yapmak isteseydik her ayrı veri formatı için özelleşmiş bir dönüşüm kodu yazmamız gerekirdi. Bu yaklaşım sonucunda filtre sayısı arttıkça kodun karmaşıklığıda gereksiz yere artmaya başlardı; hatta dönüştürme yapan kodun boyutu asıl işi yapan kodun boyutunu bile geçebilirdi. Kendimiz JSON dönüşüm kodu yazmak zorunda kalmadan değişim yapmanın bir yoluda daha kısa koda sahip olacak ama arka tarafta karmaşıklığı daha da arttıracak JSON -> XML -> XSLT/STX -> XML -> JSON şeklinde bir dönüşüm yolu izlemek olurdu. Ancak bu yaklaşımda JSON formatında bir veriyi dönüştürmek için XML dönüştürme formatlarını kullanmak zorunda kalırdık.

Nasıl Jolt (Örnek Dönüşüm)

Jolt'un kullanımı yukarıda da bahsettiğimiz gibi 3. bir JSON dosyasında verdiğimiz dönüşüm tanımları/ kuralları kullanılarak yapılıyor. Süreci requested.json = Jolt(input.json, transform.json) olarak düşünebilirsiniz.

Örnek veri olarak, elimizde filmlerle ilgili şöyle bir JSON bulunduğunu düşünelim:

{
    "movies": {
        "The Godfather": {
            "imdb": 92,
            "rotten": 98
        },
        "Goodfellas": {
            "imdb": 87,
            "rotten": 96
        },
        "Apocalypse Now": {
            "imdb": 85,
            "rotten": 96
        }
    }
}

Bu veriyi alıp, filmlerin rotten tomatoes skorunu çıkardıktan sonra, kökdeki gereksiz "movies" objesinden kurtulup, filmlere bir available: true değeri vereceğiz. Böylece aynı veride Jolt'un en yaygın 3 farklı transformunu kullanmış olacağız (Sırasıyla: remove, shift, default). Ayrıca wildcard (*) ve path operatörleride (&) karşımıza çıkacak. Örnek transform JSON'ında okuyucuya yardımcı olması için // ile yorumlar var; ancak yorum koyma özelliği JSON'ın resmi bir parçası değil, o yüzden bu örneği denerken yorumları çıkarmak lazım.

[ // json kökü liste, obje değil
    {
        "operation": "remove", // remove operasyonu yap
        "spec": { // spec o operasyona özel argümanları sağladığımız yer
            "movies": {
                "*": { // movies içindeki her objenin altında
                    "rotten": "" // rotten'ı boş tanımladığımız için sil
                }
            }
        }
    },
    {
        "operation": "shift", // shift operasyonu yap
        "spec": {
            "movies": {
                "*": { // yine movies içindeki her objeyi seç
                    // imbd objesini json içinde "&1.imdb" konumuna koy
                    // &1 burada bir üstteki objeyi temsil ediyor
                    "imdb": "&1.imdb" // örneğin: "Goodfellas.imdb"
                }
            }
        }
    },
    {
        "operation": "default", // default operasyonu yap
        "spec": {
            "*": { // artık movies'e gerek yok
                "available": true // available: true objesini ekle
            }
        }
    }
]

Bu dönüşüm sonucu elde ettiğimiz JSON şu şekilde olacak:

{
    "The Godfather": {
        "imdb": 92,
        "available": true
    },
    "Goodfellas": {
        "imdb": 87,
        "available": true
    },
    "Apocalypse Now": {
        "imdb": 85,
        "available": true
    }
}

Fark etmiş olabilirsiniz ki, spec.json'da koyduğumuz kurallar, bir obje yerine liste halinde bulunmakta. Bunu akılda tutmak önemli çünkü kurallarımız listedeki sıralarına göre işleniyor. Birden çok işlem yaparken sırayı değiştirmek beklenmeyen sonuçlara yol açabilir.

Bu örneği (veya herhangi bir Jolt dönüşümünü) Jolt'un kendi transform demo sitesinde kendiniz de deneyebilirsiniz.
